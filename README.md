# Projet-11-Machine-Learning-sur-Python
Utilisation du Machine Learning pour l'identification de faux billets
Test de quatre algorithmes de Machine Learning sur Python : 
- Régression logistique
- KNN
- Clustering
- Random Forest

Ces algorithmes ont été testés sur notre jeu de données initial et les indicateurs tels que le score de rappel, de précision et des erreurs au moindre carré ont été évalués
